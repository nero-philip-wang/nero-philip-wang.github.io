# 如何开通 Github Page

使用 Github Page 搭建博客，非常方便，同时又是免费的，只是可能国内不能访问。

其实开通 Github Page 很简单，只要你建立一个和自己用户名相同的仓库就行了，比如我的个人中心的网址是 https://github.com/nero-philip-wang (opens new window) ，那我就要建立一个nero-philip-wang/nero-philip-wang.github.io的仓库，然后就可以用 https://nero-philip-wang.github.io (opens new window) 来访问你的页面了
